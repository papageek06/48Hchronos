## Contexte
Ce projet a été réalisé dans le cadre d'un challenge de 48h. L'objectif était de concevoir un jeu interactif pour une équipe adverse.

## Explication du jeu
Le joueur débute sur une page de navigation et doit explorer différentes pages pour trouver les outils qui vont lui permettre d'ajuster l'évier. Pour progresser, il doit résoudre des énigmes. Une vague d'eau monte progressivement sur la page, faisant office de chronomètre. Le joueur doit atteindre la page du combat final à temps, sinon il se noie. Le canard, d'abord amical, devient de plus en plus hostile au fil de l'aventure, craignant de perdre son habitat aquatique.

## Réponses / Codes

- **Page de navigation**
  
	*Contexte :* Le joueur commence son aventure sur une page centrale. Il doit explorer et deviner des mots-clés pour accéder aux différentes épreuves du jeu.
  
	*But :* Trouver les bons mots pour débloquer l'accès aux autres pages et progresser dans l'histoire.

	*Solution :* (à compléter selon les mots utilisés dans le jeu)

- **Site de plomberie**
  
	*Contexte :* Le joueur se retrouve sur un faux site de plomberie. Il doit acheter un outil pour réparer l'évier, mais un code spécial est nécessaire pour valider la commande.
  
	*But :* Explorer la page 404 pour trouver le code caché, puis l'utiliser pour finaliser l'achat dans le panier.

	*Solution :* `Canard2026`

- **Page de cadenas**
  
	*Contexte :* Une énigme sous forme de cadenas bloque l'accès à un nouvel outil. Le joueur doit résoudre l'énigme pour obtenir la bonne combinaison.
  
	*But :* Trouver la suite de chiffres correcte pour ouvrir le cadenas et récupérer un outil supplémentaire.

	*Solution :* `14 15 1 8 7`

- **Tableau périodique**
  
	*Contexte :* Grâce aux chiffres trouvés précédemment, le joueur doit reconstituer un mot en s'aidant d'un tableau périodique.
  
	*But :* Décoder le mot secret pour obtenir le dernier outil nécessaire à la réparation.

	*Solution :* `siphon`

- **Combat final**
  
	*Contexte :* Le joueur affronte le canard devenu hostile dans un combat décisif. Un pistolet apparaît pour l'aider à vaincre son adversaire.
  
	*But :* Battre le canard pour stopper la montée de l'eau et remporter la partie.

	*Solution :* Utiliser le pistolet qui apparaît pour vaincre le canard.
